Predicting House Prices with Linear Regression

This project focuses on predicting house prices using **Linear Regression**, one of the most fundamental and widely used Machine Learning algorithms for regression problems.

The main goal of this project is to understand how different factors such as area, number of bedrooms, bathrooms, parking availability, and furnishing status influence the price of a house. By analyzing the dataset and training a regression model, we can make accurate price predictions based on given property features.

 Project Overview

In this notebook, I performed:

* Data loading and exploration
* Data cleaning and preprocessing
* Handling categorical variables
* Exploratory Data Analysis (EDA)
* Feature selection
* Splitting dataset into training and testing sets
* Training a Linear Regression model
* Model evaluation using performance metrics
* Visualization of results and predictions

This project helped me gain practical experience in:
